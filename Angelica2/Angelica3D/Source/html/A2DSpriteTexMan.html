A2DSpriteTexMan

Докладний опис:

Клас A2DSpriteTexMan відповідає за керування всіма текстурами для 2D-спрайтів у рушії Angelica3D. Він виконує завантаження, кешування, ініціалізацію, пошук, видалення та звільнення текстур, а також керує потоками вершин для рендеру спрайтів. Клас також працює з шейдерами для спеціальних ефектів.

Основні функції:

  • Ініціалізація потоків вершин і шейдерів для рендеру спрайтів.
  • Завантаження текстур з файлів та з пам'яті, кешування та повторне використання.
  • Звільнення текстур, очищення кешу, скидання стану.
  • Пошук текстур за ім'ям файлу.
  • Керування буферами для рендеру спрайтів з однією або двома текстурними шарами.

Структура даних:

  • m_pA3DDevice — пристрій рендерингу (A3DDevice).
  • m_pStream — потік вершин для звичайних спрайтів.
  • m_pStream2Layer — потік вершин для спрайтів з двома шарами текстур.
  • m_pStereoShader — шейдер для стереоефектів.
  • m_ListTexture — список усіх завантажених текстур (TEXTURERECORD).
  • m_CachedTextures — кешовані текстури для швидкого доступу.
  • m_nMaxBlades, m_nMaxVerts, m_nVertsCount — параметри для керування кількістю вершин у буферах.

Типові методи:

  • Init — ініціалізація менеджера текстур.
  • Release — звільнення всіх ресурсів.
  • Load2DSpriteTexturesFromFile — завантаження текстур з файлу.
  • Release2DSpriteTexture — звільнення конкретної текстури.
  • Reset — скидання стану менеджера.
  • FlushInternalBuffer — очищення внутрішнього буфера.
  • LockStream/UnlockStream — блокування/розблокування буфера для рендеру.
