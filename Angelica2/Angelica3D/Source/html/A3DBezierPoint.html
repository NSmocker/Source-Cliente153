A3DBezierPoint

Докладний опис:

Клас A3DBezierPoint представляє точку Безьє, яка використовується для побудови кривих Безьє у 3D-просторі. Клас відповідає за зберігання позиції, кольору, радіусу точки, а також за її рендеринг, вибір (підхоплення) та оновлення геометрії. Застосовується для редагування та візуалізації кривих у редакторах або ігрових сценах.

Основні функції:

  • Зберігання позиції, кольору та радіусу точки Безьє.
  • Рендеринг точки у вигляді простого полігону.
  • Підтримка вибору точки (Pick) для інтерактивного редагування.
  • Оновлення геометрії точки при зміні параметрів.

Структура даних:

  • m_pA3DDevice — пристрій рендерингу.
  • m_vecPos — позиція точки у 3D-просторі.
  • m_color — колір точки.
  • m_vRadius — радіус точки.
  • m_pVertices — масив вершин для рендеру полігону.
  • m_pIndices — індекси для рендеру полігону.
  • m_aabb — Axis-Aligned Bounding Box для вибору точки.

Типові методи:

  • Init — ініціалізація точки.
  • Release — звільнення ресурсів.
  • Render — рендеринг точки.
  • Pick — перевірка, чи вибрана точка променем.
  • SetColor, SetPosition, SetRadius — зміна параметрів точки.
  • UpdatePoint — оновлення геометрії точки.
