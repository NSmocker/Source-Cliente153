Діаграма класів (A3DSDK)

A3DMeshBase
A3DSkinMesh
SUBMESH
SKINMESHDATA

Пояснення класів

  • A3DMeshBase — Базовий клас для всіх 3D-мешів у рушії. Містить основні атрибути та методи для роботи з 3D-геометрією.
  • A3DSkinMesh — Клас, що наслідує A3DMeshBase і реалізує скінинг (анімовану сітку з кістками). Містить методи для завантаження, рендерингу, обробки вершин, роботи з підсітками та матеріалами.
  • SUBMESH — Внутрішня структура A3DSkinMesh, що описує підсітку: набір вершин, індексів та матеріалів для частини сітки.
  • SKINMESHDATA — Структура для збереження/завантаження даних скіна: індекси текстур, матеріалів, кількість вершин та індексів.

Інші допоміжні класи

  • A3DStream — Клас для роботи з потоками даних (vertex/index buffer).
  • A3DTexture — Клас для представлення текстури у рушії.
  • A3DMaterial — Клас для опису матеріалу (властивості поверхні).

Діаграма показує ієрархію та зв'язки основних класів, які визначають структуру та функціонал скінажної сітки у рушії A3D.
